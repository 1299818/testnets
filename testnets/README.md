# IRIShub 测试网

## IRIShub 简介

IRIS Hub是在Cosmos生态中的区域性枢纽，提供iService服务

## 安装IRIShub

### 方法1：下载发行版安装

进入下载页: https://github.com/irisnet/irishub/releases/
下载对应版本的可执行文件
解压缩tar -C /usr/local/bin -xzf iris$VERSION.$OS-$ARCH.zip
拷贝到/usr/local/bin/目录下 
执行以下命令,若出现对应的版本号则说明安装成功。
```
$ iris version
v0.2.0
    
$ iriscli version
v0.2.0
```
